# CANSLIM Screen — 成长股基本面+技术面筛选

基于欧奈尔对1880年以来美国股市1000多只超级牛股的系统研究，总结出七个共同特征。
核心逻辑：**找到业绩正在加速增长的公司，在技术面突破时买入。**

欧奈尔的"超级悖论"：那些对大多数人来说价格过高、风险过大的股票，往往会百尺竿头更进一步；而那些看似低价便宜的股票却常常进一步走低。

---

## CANSLIM 七要素

### C — Current Quarterly Earnings（当季盈利增长）

**标准：当季EPS同比增长 ≥ 25%，越高越好（40%～500%更佳）**

**关键细节（原书第3章）：**
- 研究表明：1952-2001年表现最好的600只股票，75%在大涨前最新季报EPS平均上涨70%以上
- 加速增长比稳定增长更重要：连续几季度增速提升（如20%→25%→35%→50%）
- 销售收入也应同步增长（至少 ≥ 25%），否则盈利增长不可持续
- 排除一次性收益（资产出售等非经营性收益）
- 连续两个季度收益大幅回落（降至原增速1/3以下）是危险信号

**市盈率的误区（原书第4章）：**
- 欧奈尔明确反对依赖市盈率选股
- 1953-1985年表现最佳股票初始平均市盈率20倍，最终涨至45倍（+125%）
- 1990-1995年领军股初始市盈率36，最终涨至80多
- 结论：不要因为市盈率高就回避，也不要因为市盈率低就买入

---

### A — Annual Earnings Growth（年度盈利增长）

**标准：过去3年年均EPS增长 ≥ 25%，ROE ≥ 17%**

**关键细节（原书第4章）：**
- 1980-2000年研究：表现优异股票早期平均年度收益增长率为36%
- 3/4的强势股在股价飙升前3-5年都有可观的年度收益增长
- 收益稳定性数值（1-99）应低于20-25，说明增长稳定且持续
- 最近几个季度的收益增长应有所加快（加速增长）
- 周期型/逆转型股票：年度增长率至少5%-10%，连续两季度强势回升

---

### N — New（新公司、新产品、新管理层、股价新高）

**三个"新"之一即可，且股价应处于突破新高状态（原书第5章）**

**关键细节：**
- 新产品/新服务：能带来革命性变化的创新（非改良）
- 新管理层：带来战略转型的新领导层
- **价格创新高（最重要且最反直觉）**：
  - 欧奈尔研究：买入正在创新高的股票，而非从最高点大幅回落的股票
  - 真正的大牛股都是从新高出发继续创新高
  - 从50美元涨到100美元，必须经历51、52...等一系列新高

**买入时机：** 从有效价格形态中突破出来时，成交量放大至均量150%以上

---

### S — Supply and Demand（供给与需求）

**关注流通股规模和成交量变化（原书第6章）**

**供给面：**
- 优先选择流通股较少的公司（小盘股 < 2500万股更佳）
- 公司积极回购是加分项（10%回购属大规模）
- 管理层持股比例高是加分项（大公司≥1-3%，小公司更高）
- 避免过度分拆（1:3或1:5的分拆比例会稀释供给）

**需求面：**
- 突破时成交量应放大至均量**150%以上**（最好200%+）
- 上涨日成交量大、下跌日成交量小 → 机构在买入
- 上涨日成交量小、下跌日成交量大 → 机构在出货

---

### L — Leader or Laggard（领军股或落后者）

**核心：只买行业前三甲，不买跟随者（原书第7章）**

**相对强度（RS）评级：**
- IBD评级体系：1-99，99为最强
- **标准：RS ≥ 80，越高越好（真正大牛股突破前RS通常 ≥ 90）**
- 研究数据：1952-2008年表现最好的股票，大涨前平均RS为87
- 不买RS低于70的股票，即使基本面好也要谨慎

**重要洞见：**
- 不要买"跟随性投资"（同行业中比领军股便宜的股票）
- 案例：先达（领军）vs 西尔制药（跟随）→ 先达涨400%，西尔几乎没动
- 大盘调整时跌幅最小的股票，往往是下一轮的领军股

---

### I — Institutional Sponsorship（机构持股）

**核心：有质量机构持股且在增加（原书第8章）**

**质量判断：**
- 至少有几家机构持股（最少20家）
- 最近几个季度机构持有者数量在增加
- 最近一季度有新的优质机构（近期业绩A级或B+级基金）买入
- 避免"过量持有"：机构太多说明上涨空间有限，潜在抛压大

**警戒信号：**
- 机构数量在减少（聪明钱在离场）
- 1000多家机构持有同一只股票（如2000年的思科、美国在线）→ 过量持有

---

### M — Market Direction（市场方向）

**最重要的要素：75%的股票跟随大盘涨跌（原书第9章）**

**牛市判断（可以积极做多）：**
- 大盘指数处于上升趋势
- 上涨日成交量 > 下跌日成交量
- 新高股票数 > 新低股票数
- 领军股表现强劲

**出货日（Distribution Day）识别：**
- 大盘指数下跌超过0.2%且成交量放大
- **4-6周内出现3-5次出货日 → 市场可能见顶**
- 成交量增加但指数涨幅很小，也算出货

**跟随日（Follow-Through Day）识别（牛市开始信号）：**
- 大盘从底部反弹后第4天起
- 某天出现放量大涨（涨幅 ≥ 1.7%，成交量大于前一天）
- 这是可以开始重新买入的初步信号

## 图表形态（买入时机）

### 带柄茶杯形态（最经典）

```
价格走势：
     ___         ___
    /   \       /
   /     \_____/
  /         ^
 /         柄部（回调5-15%，成交量萎缩）
```

**参数要求（原书第2章）：**
- 杯形：7-65周，深度12%-33%（牛市中调整幅度为大盘的1.5-2.5倍）
- 柄部：至少1-2周，回调8%-12%（牛市），成交量萎缩，位于形态上半部
- **中轴买入点**：柄部最高价上方0.05-0.10美元
- **成交量**：突破时放大至均量150%以上（最好200%+）

**警戒信号（有缺陷的形态）：**
- 柄部在形态下半部（弱势）
- 柄部楔形向上（弱势）
- 杯形过深（> 45%）
- 突破时成交量不足

### 双重底形态（"W"形）

- 第二个底部应略低于第一个（淘汰弱手）
- 中轴买入点在W中间最高点处
- 两个底部之间必须有明确的反弹

### 平底形态

- 出现在带柄茶杯形态后上涨20%以上
- 5-7周横向整理，回调不超过10-15%
- 成交量萎缩

### 高而窄旗形（罕见但强势）

- 4-8周内猛涨100%-120%，然后回调不超过10%-25%
- 非常罕见，风险大，但一旦成功涨幅惊人

## 卖出规则

### 止损（铁律，无任何例外）

**买入后下跌7-8%立即止损**

- 不要"再等等看"
- 不要"平均成本"
- 不要等到收盘再行动
- 如果熊市中买入，止损标准收紧到3%-4%

**逻辑：** 正确买入时机的股票不应在买入后立即下跌7-8%；超出这个界限说明判断错误或大盘要下跌

### 主动止盈（20%-25%原则）

- 股价上涨20%-25%后考虑卖出获利
- **例外**：如果股票在1-3周内急涨20%，至少持有8周（超级强势股）
- 3笔20%的利润 = 年化约44%收益

### 被动卖出信号（顶峰信号）

**顶峰前的疯狂表现：**
- 连续2-3周急速上涨（日线图上8-10天中有7-8天强势上涨）
- 出现消耗性缺口（大成交量跳空高开）
- 最大的单日成交量（可能是股价顶点）
- 股票分拆后1-2周内仍上涨25%-50%

**其他卖出信号：**
- 当季收益增长率连续两季度降至原增速1/3以下
- 股价跌破50日均线且成交量放大
- RS评级降至70以下
- 第三、四阶段价格形态突破（失败率80%）
- 成交量萎缩情况下创价格新高（需求减弱）

## 与 elder-screen 的配合

```
完整分析流程：

1. canslim-screen → 确认基本面质量（值不值得关注）
   - 评分 ≥ 5/7，尤其C和A要满足 → 进入下一步
   - 评分 < 5/7 → 放弃，寻找更好的标的

2. elder-screen → 确认技术面时机（现在是不是好时机）
   - 三重滤网确认 → 可以买入
   - 信号冲突 → 等待

3. position-sizer → 计算仓位（2%/6%原则 + R倍数）
```
