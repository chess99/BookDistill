# CANSLIM Screen — 成长股基本面+技术面筛选

CANSLIM 是欧奈尔研究了数十年大牛股后总结的选股框架。
核心逻辑：**找到业绩正在加速增长的公司，在技术面突破时买入。**

---

## CANSLIM 七要素

### C — Current Quarterly Earnings（当季盈利增长）

**标准：当季EPS同比增长 ≥ 25%**

- 越高越好，最优秀的股票往往增长50%、100%甚至更高
- 加速增长更好：连续几个季度增速在提升
- 排除一次性收益（资产出售等非经营性收益）
- 销售收入也应同步增长（至少 ≥ 25%），否则盈利增长不可持续

**警戒信号：**
- 盈利增长来自裁员降本，而非收入增长
- 最近一季增速明显放缓

---

### A — Annual Earnings Growth（年度盈利增长）

**标准：过去3年年均EPS增长 ≥ 25%，且最近一年增长最快**

- 寻找有持续增长记录的公司，而非昙花一现
- ROE（净资产收益率）≥ 17%，说明公司有效利用股东资金
- 利润率稳定或在提升

**理想模式：** 年增长率从20%→25%→35%→50%（加速增长）

---

### N — New（新产品/新管理层/新高）

**三个"新"之一即可：**

1. **新产品或新服务**：能显著提升公司收入的创新（iPhone、云计算等）
2. **新管理层**：带来战略转型的新领导
3. **价格新高**：股价突破历史新高（反直觉但极重要）

**关键洞见：** 大多数人害怕在新高买入，但真正的大牛股都是从新高出发继续创新高。
欧奈尔说："不要买便宜货，要买真正的强者。"

---

### S — Supply and Demand（供需关系）

**两个维度：**

**股本规模（Supply）：**
- 优先选择流通股较少的公司（< 2500万股更佳）
- 小盘股更容易因需求增加而大幅上涨
- 大盘股（> 5亿股）需要巨量资金才能推动，上涨更难

**成交量（Demand）：**
- 突破时成交量应放大至均量的 **150%以上**（最好200%+）
- 上涨时成交量大，下跌时成交量小 → 机构在买入
- 上涨时成交量小，下跌时成交量大 → 机构在出货

**公司回购：** 积极回购股票的公司是加分项（减少供给）

---

### L — Leader or Laggard（领袖还是落后者）

**标准：相对强度（RS）≥ 80（IBD评级体系）**

- 只买行业龙头，不买落后者
- 在同行业中选择表现最强的1-2只
- 即使基本面好，但股价表现落后于大盘的股票也要回避

**实用方法：**
- 比较同行业股票，选择过去52周涨幅最大的
- 在大盘下跌时，领袖股跌幅明显小于大盘（相对强势）
- 大盘反弹时，领袖股率先创新高

---

### I — Institutional Sponsorship（机构持股）

**标准：有质量机构持股，但不要过度拥挤**

- 需要有机构持股（基金、保险等），因为机构买入才能推动价格
- 但持有机构数量不要太多（过度拥挤意味着未来抛压大）
- 理想状态：机构数量在增加（说明越来越多聪明钱在买入）
- 最近一两个季度机构持股比例在上升

**警戒信号：**
- 机构数量在减少（聪明钱在离场）
- 持有同一只股票的机构太多（大家都知道了，上涨空间有限）

---

### M — Market Direction（市场方向）

**这是最重要的要素：** 75%的股票跟随大盘涨跌

**牛市特征（可以积极做多）：**
- 大盘指数（沪深300/纳斯达克）处于上升趋势
- 上涨日成交量 > 下跌日成交量
- 新高股票数 > 新低股票数
- 大盘在关键均线（50日/200日）之上

**熊市特征（应减少持仓或观望）：**
- 大盘连续出现"出货日"（指数下跌超0.2%且成交量放大）
- 4-6周内出现4-5次出货日 → 市场可能见顶
- 领袖股开始跌破支撑位

**操作原则：** 在熊市中，即使找到完美的CANSLIM股票也应谨慎，等待大盘确认转强后再买入。

---

## 买入时机：图表形态

### 杯柄形态（Cup with Handle）

最经典的买入形态：

```
价格走势：
     ___         ___
    /   \       /
   /     \_____/
  /         ^
 /         柄部（回调5-15%，成交量萎缩）
/
```

- **杯形**：通常历时7周到65周，深度15-35%
- **柄部**：回调幅度不超过杯深的1/3，成交量萎缩
- **买入点（枢轴点）**：柄部最高点上方 0.05-0.10 元
- **成交量**：突破时放量至均量150%以上

### 平底形态（Flat Base）

- 价格在5-7周内横向整理，回调不超过15%
- 成交量萎缩
- 在上方突破时买入

### 双底形态（Double Bottom）

- 两个低点，第二个低点略高于第一个
- 中间的反弹高点是关键
- 在第二个低点回升并突破中间反弹高点时买入

---

## 卖出规则

### 止损（保护资本）

**铁律：买入后下跌7-8%立即止损，无条件执行**

- 不要"再等等看"，不要"平均成本"
- 这是欧奈尔最强调的规则

### 止盈（锁定利润）

**主动止盈：**
- 上涨20-25%后，考虑卖出2/3仓位
- 如果是真正的领袖股且大盘强势，可以持有更久

**被动止盈信号（卖出信号）：**
- 股价在高位出现大成交量下跌（出货信号）
- 股价跌破50日均线且成交量放大
- 基本面恶化（盈利增长放缓至 < 15%）
- 大盘进入熊市（出现多次出货日）

```
# CANSLIM 分析报告 — [股票名称/代码]

## 评分总览
[C][A][N][S][L][I][M] 各项评分（✅满足 / ⚠️部分满足 / ❌不满足）
综合评分：[X/7]

## 详细分析

### C - 当季盈利
- 最近一季EPS同比增长：[X%]
- 评价：[满足/不满足标准（≥25%）]
- 销售收入增长：[X%]

### A - 年度盈利
- 近3年年均增长：[X%]
- ROE：[X%]
- 评价：[加速增长/稳定增长/放缓]

### N - 新要素
- [新产品/新管理层/价格新高 的具体描述]

### S - 供需
- 流通股数量：[X亿股]
- 近期成交量趋势：[放量上涨/缩量下跌/...]
- 评价：[机构在买入/出货/中性]

- 行业排名：[前X名]

[具体的买入/观望/回避建议，以及与 elder-screen 配合使用的建议]

```

## 与 elder-screen 的配合

```
完整的分析流程：

1. canslim-screen → 确认基本面质量（值不值得关注）
   - 评分 ≥ 5/7 → 进入下一步
   - 评分 < 5/7 → 放弃，寻找更好的标的

2. elder-screen → 确认技术面时机（现在是不是好的入场时机）
   - 三重滤网确认 → 可以买入
   - 信号冲突 → 等待

3. position-sizer → 计算仓位
   - 按2%/6%原则确定买入量
```
